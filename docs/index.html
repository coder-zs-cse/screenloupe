What it does

Built for people who pay attention to detail.

System-wide

Works in VS Code, Terminal, Settings, File Explorer, games — anything on your screen, not just the browser.

Configurable trigger

Middle-click (smart tap-through), side mouse buttons, Right Ctrl, Right Shift — pick whichever fits your workflow.

Circle or strip

A round loupe for precision or a full-width rectangular strip — great for reading code across the whole screen.

Native GPU acceleration

Powered by the Windows Magnification API — the same engine as the OS Magnifier. No screenshot hacks, no lag.

No Python required

The installer bundles everything. One .exe, double-click, done. No dependencies to wrangle.

Open source

MIT licensed. The whole thing is a few hundred lines of Python. Read it, fork it, improve it.

Install

Two ways in.

The installer is for everyone. The CLI is for people who already have Python and just want it running.

ScreenLoupe-Setup.exe latest release ↗

Windows 10/11 · ~12 MB · No Python needed · All releases

What happens on install: copies the exe to Program Files, adds a Start Menu entry, and optionally sets it to run silently at login (checkbox is checked by default). You can change all settings later by searching "ScreenLoupe" in the Start menu.
With Python (pipx)
Install pipx recommended
# one-time: install pipx if you don't have it
pip install pipx

# install ScreenLoupe from GitHub
pipx install git+https://github.com/coder-zs-cse/screenloupe
Launch
# open the settings window
screenloupe

# run silently in background
screenloupe --background
Configure
screenloupe-debug config set zoom 4
screenloupe-debug config set trigger rctrl
screenloupe-debug config set shape rect
